Privacy Policy

Last updated: [Fecha]

AI Gastos ("the Extension") respects your privacy and is committed to protecting your information.

1. Information We Access

With the user's explicit authorization, the Extension may access:

  • Email content from Gmail messages related to financial transactions
  • Transaction-related financial data (such as merchant name, amount, date)
  • Google account email address (for authentication)
  • Google Sheets identifiers selected by the user

The Extension does not access passwords, authentication credentials, unrelated emails, or browsing history.

2. How We Use the Information

The accessed information is used exclusively to:

  • Detect transaction-related emails
  • Extract structured expense information
  • Categorize transactions using AI-based processing
  • Sync the structured expense data into a Google Sheet selected by the user

The data is used solely to provide the core functionality of the Extension.

3. AI Processing

To extract and categorize transaction information, relevant portions of email content may be securely transmitted to a backend service controlled by the developer.

The backend service may use OpenAI's API to process and structure the data.

The data is:

  • Used only for processing the user's request
  • Not sold or shared for advertising purposes
  • Not used for profiling unrelated to the Extension's functionality

No personal data is intentionally stored beyond what is necessary to provide the requested service.

4. Data Sharing

The Extension does not sell, rent, or trade user data.

Data is transmitted only to:

  • Google APIs (Gmail and Google Sheets)
  • The developer's backend processing service
  • OpenAI API strictly for structured data extraction

All transmissions occur over secure HTTPS connections.

5. Data Storage

User preferences are stored locally using Chrome storage.

Personal email content is not permanently stored unless required for core functionality.

6. User Control

Users may:

  • Revoke Google permissions at any time
  • Uninstall the Extension
  • Contact the developer for data deletion requests

7. Contact
